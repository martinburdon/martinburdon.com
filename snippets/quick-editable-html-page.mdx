Open up a quick editable page in browser to take down throw away notes

## Snippet

```js
data:text/html,<body contenteditable style="line-height:1.5;font-size:20px;">
```

## Usage

Open up a new tab and paste the snippet into the URL bar. You should also be able to apply bold and italic styles with `Ctrl + B` and `Ctrl + i`
